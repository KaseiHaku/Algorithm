################################ 决策(Decision) 算法 ################################
线性规划: 坐标系中，选择多个 线性函数 围成的 范围 内的 点

层次分析法(AHP): 制定一系列 指标， 每个指标带权重(1,3,5,7,9)，针对每个 选择 都计算每个指标的得分，选出总得分最高的

TOPSIS(优劣解距离法): 要求每个 指标函数 都是 单调函数


################################ 其他算法 ################################
灰色关联分析：适用于判断两个指标的关联度

信息熵权重法：适用于已有数据，需要 定权，即: 计算打分标准
    原始统计数据下各个指标得分:
        A   B   C   D   E
        A1  B1  C1  D1  E1
        A2  B2  C2  D2  E2
    标准化:
        YA = An - min(A) / max(A) - min(A)          正向指标时，公式
        YA = max(A) - An / max(A) - min(A)          负向指标时，公式
    
    
    信息量： 
        定义: 一个具体事件发生了所带来的信息。这里还可以发现一个事实：如果某一事件的概率为 100%，那么其信息量为 0，即"已经确定的信息没有传递的价值"
        公式: h(x) = -log₂P(x),   P(x) = 事件 x 发生的概率 [0,1]
    信息熵(单位: bit 比特)：
        公式: H(X) = p(x1)*h(x1) + p(x2)*h(x2) + ...
        
    权重计算公式: w = 1 - H(x) / k - ∑H(X)       其中 k 是指标的个数(A-E) 5 个指标 
    
支持向量机：适用于分类
